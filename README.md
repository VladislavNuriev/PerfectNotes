# Приложение для создания заметок

## Описание

Приложение позволяет:
- создавать и сохранять текстовые заметки;
- осуществлять поиск по существующим заметкам;
- закреплять/откреплять заметки;
- удалять заметки.

Демонстрация интерфейса приложения:

## Использованные технологии

- **Jetpack Compose** — для построения UI;
- **MVI + MVVM** — архитектурный паттерн;
- **Coroutines Flow** — для асинхронной обработки данных;
- **Room** — для локального хранения данных;
- **Hilt** — для внедрения зависимостей;
- **Jetpack Navigation** — для навигации между экранами;
- **Многомодульность** — организация структуры проекта.
